## 📊 常用命令

### 快速分析

| 命令 | 说明 |
|------|------|
| `python3 analyze.py 601138` | 自动检测图形类型 |
| `python3 analyze.py 601138 -p 开盘跳水` | 指定图形类型 |
| `python3 analyze.py 601138 -p auto` | 显式启用自动检测 |

### 图形类型

| 类型 | 代码 |
|------|------|
| 开盘跳水 | `-p 开盘跳水` |

| 冲板回落 | `-p 冲板回落` |

## 🔍 系统自动识别的图形

### ✅ 会分析（使用AI模板）

| 图形 | 条件 | 示例 |
|------|------|------|
| 开盘跳水 | 开盘后跌超2% | 跌幅2.5% |
| 冲板回落 | 冲高超8%且回落超3% | 冲高9%回落5% |
| 破位下跌 | 跌破支撑位 | 跌破20日线 |

### ⚠️  不会分析（给出建议）

| 状态 | 条件 | 系统响应 |
|------|------|---------|
| 强势上涨 | 涨超5%或涨停 | 提示关注止盈 |
| 震荡整理 | 涨跌幅±2%以内 | 建议观望 |

```bash

# 禁用自动检测
python3 analyze.py 601138 -p 开盘跳水 --no-auto-detect
```

## 📱 实用技巧

### 技巧1：创建分析脚本

```bash
# 创建 my_stocks.sh
#!/bin/bash
for code in 601138 600519 002594; do
    python3 analyze.py $code
    echo ""
done
```

### 技巧2：定时分析

```bash
# 使用crontab定时分析
# 每天上午9:35分析
35 9 * * 1-5 cd /path/to/stock && python3 analyze.py 601138
```

### 技巧3：保存分析结果

```bash
# 保存到文件
python3 analyze.py 601138 > analysis_$(date +%Y%m%d_%H%M).txt
```
